Data engineer

Job brief


You will use various methods to transform raw data into useful data systems. Overall, you will strive for efficiency by aligning data systems with business goals. Data engineer skills also include familiarity with several programming languages and knowledge of learning machine methods.

Key responsibilities


  • Analyze and organize raw data
  • Build data systems and pipelines
  • Evaluate business needs and objectives
  • Interpret trends and patterns
  • Conduct complex data analysis and report on results
  • Prepare data for prescriptive and predictive modeling
  • Build algorithms and prototypes

Skills and qualifications for this position:


Essentials

  • Previous experience as a data engineer or in a similar role
  • Technical expertise with data models, data mining, and segmentation techniques
  • Knowledge of programming languages (e.g. Java and Python)
  • Hands-on experience with SQL database design

Preferable

  • Great numerical and analytical skills
  • Degree in Computer Science, IT, or similar field; a Master degree is a plus
  • Data engineering certification (e.g IBM Certified Data Engineer) is a plus
